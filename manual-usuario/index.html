1 Introducción al Sistema

Conozca las capacidades y beneficios de SAM Metrología

¿Qué es SAM Metrología?

SAM (Sistema de Aseguramiento Metrológico) es una plataforma web especializada en la gestión integral de equipos de medición, diseñada para cumplir con los requisitos de las normas ISO 17020:2012 e ISO 17025:2017.

Trazabilidad Completa

Registre y consulte todo el historial de calibraciones, mantenimientos y comprobaciones

Alertas Automáticas

Reciba notificaciones antes del vencimiento de calibraciones y mantenimientos

Análisis de Tendencias

Monitoree el desempeño de sus equipos con gráficas y estadísticas

Documentación ISO

Genere reportes conforme a normas internacionales de metrología

Beneficios Clave
  • Ahorre tiempo con automatización de tareas metrológicas
  • Cumpla requisitos normativos ISO 17020 e ISO 17025
  • Acceda desde cualquier lugar con conexión a internet
  • Proteja sus datos con seguridad multiempresa

Módulos Principales

Módulo Funcionalidad Norma
Equipos Gestión completa del inventario de equipos de medición ISO 10012:2003
Calibraciones Registro y trazabilidad de calibraciones externas ISO 17025:2017
Confirmación Análisis de aptitud metrológica post-calibración ISO 10012:2003
Comprobación Verificaciones intermedias entre calibraciones ISO 17020:2012
Mantenimiento Programación y seguimiento de mantenimientos ISO 10012:2003
Reportes Generación de PDFs, Excel y documentación oficial ISO 17020/17025

3 Creación de Equipos

Registre nuevos equipos de medición en el sistema

Datos Básicos del Equipo

Para registrar un equipo vaya a Equipos → Nuevo Equipo y complete los campos requeridos:

Campo Descripción Obligatorio
Nombre Nombre descriptivo del equipo (ej: "Balanza analítica Mettler AB204") Sí
Código / Etiqueta Código interno único del laboratorio (ej: BAL-001) Sí
Número de Serie Serie del fabricante para trazabilidad No
Magnitud / Familia Tipo de medición (Masa, Longitud, Temperatura…). Se selecciona del catálogo. Sí
Ubicación Área o laboratorio donde está físicamente el equipo No
Intervalo de Calibración Cada cuántos meses se debe calibrar (el sistema calcula la próxima fecha) Sí

Catálogos: Magnitudes y Familias

El catálogo define las magnitudes (Masa, Longitud, Temperatura, Presión…) y las familias de equipos disponibles en su organización. Un administrador puede configurarlos en Configuración → Catálogos.

¿Por qué son importantes los catálogos?
  • Unifican la terminología de toda la organización
  • Permiten filtrar y agrupar equipos por tipo de medición
  • Son la base para definir los puntos de medición en confirmaciones y comprobaciones
  • Facilitan la generación de reportes por magnitud (ISO 10012)
1
Vaya a Configuración → Catálogos
Solo disponible para roles Administrador y Gerencia
2
Cree o edite magnitudes
Defina nombre, símbolo y unidades de medición (ej: Masa → kg, g, mg)
3
Asocie familias
Las familias agrupan equipos similares (ej: "Balanzas" pertenece a "Masa")
Consejo

Configure los catálogos antes de registrar equipos. Cambiarlos después puede afectar reportes históricos.

4 Gestión de Calibraciones

Registre y haga seguimiento a las calibraciones externas

¿Qué es una Calibración?

La calibración es el proceso de comparar las indicaciones de un equipo de medición con un patrón de referencia trazable. En SAM se registra el certificado de calibración externo emitido por el laboratorio acreditado.

Registrar una Calibración

1
Abra el detalle del equipo
Vaya a Equipos y haga clic en el equipo que fue calibrado
2
Clic en "Nueva Calibración"
Complete: fecha, laboratorio, número de certificado, resultado (Apto / No Apto)
3
Adjunte el certificado PDF
El sistema guarda el archivo y lo asocia permanentemente al registro
4
El sistema actualiza la próxima fecha
Automáticamente calcula la próxima calibración según el intervalo configurado
Equipo No Apto

Si el resultado es No Apto, el equipo queda marcado como fuera de servicio y el sistema bloquea su uso hasta que se registre una calibración satisfactoria o se dé de baja.

5 Confirmación Metrológica

Evalúe la aptitud del equipo para su uso previsto (ISO 10012)

¿Qué es la Confirmación Metrológica?

La confirmación metrológica es el proceso definido en ISO 10012:2003 que verifica si un equipo, después de calibrarse, cumple con los requisitos metrológicos del proceso de medición para el que se va a usar. No basta con tener el certificado: hay que analizar si los errores encontrados en la calibración son aceptables para la aplicación específica.

¿Qué es el EMP (Error Máximo Permitido)?

El EMP (Error Máximo Permitido, también llamado MPE en inglés) es el límite de error que un equipo puede tener y aún ser considerado apto para un uso específico. Lo define el usuario o el proceso, no el fabricante ni el laboratorio de calibración.

Ejemplo práctico:

  • Una balanza tiene un error de calibración de ±0,8 g
  • Su proceso requiere pesar con una tolerancia de ±2,0 g
  • Como 0,8 g < 2,0 g, el equipo es APTO ✓

En SAM puede definir el EMP para cada punto de medición, y el sistema calcula automáticamente si el equipo es apto, rechazado o está en zona de riesgo.

Cómo Registrar una Confirmación

1
Abra el equipo → pestaña "Confirmación"
Disponible después de registrar al menos una calibración
2
Defina las variables de medición
Para equipos con múltiples magnitudes (ej: una cámara que mide temperatura Y humedad), puede agregar una pestaña por cada variable
3
Ingrese los puntos de medición
Para cada punto: valor de referencia, valor indicado por el equipo, incertidumbre y EMP. El sistema calcula el error y determina aptitud automáticamente.
4
Genere el PDF de confirmación
El documento incluye tabla de resultados, gráfica de aptitud y decisión final firmada por el técnico responsable
Resultado Condición Acción recomendada
APTO Error ≤ EMP en todos los puntos El equipo puede usarse normalmente
ZONA DE RIESGO Error cercano al EMP (>80%) Monitorear, considerar recalibración pronto
NO APTO Error > EMP en algún punto Retirar de servicio, enviar a recalibración o reparación

6 Comprobación Intermedia

Verificaciones periódicas entre calibraciones (ISO 17020)

¿Por qué hacer comprobaciones intermedias?

La comprobación intermedia permite detectar a tiempo si un equipo ha derivado fuera de sus límites aceptables entre calibraciones. Es especialmente importante para equipos críticos o en ambientes exigentes. Está requerida por ISO 17020:2012 e ISO 17025:2017.

Periodicidad

Diaria, semanal o mensual según criticidad del equipo

Gráfica de tendencia

SAM genera automáticamente el gráfico histórico de desempeño

Registro trazable

Cada comprobación queda firmada con fecha, técnico y patrón usado

Registrar una Comprobación

1
Vaya al equipo → pestaña "Comprobación"
O use el atajo desde el calendario de actividades
2
Ingrese los valores medidos con el patrón
Registre: valor de referencia (patrón), valor indicado por el equipo y la fecha de medición
3
El sistema evalúa y genera alerta si hay desvío
Si el error supera el EMP configurado, se genera una notificación automática

8 Intervalos de Calibración

Determinación y ajuste de intervalos según ILAC G-24:2022

¿Por qué es importante el intervalo de calibración?

Un intervalo muy largo puede llevar a que el equipo derive sin detectarse. Un intervalo muy corto genera costos innecesarios. El estándar ILAC G-24:2022 proporciona métodos para determinar intervalos óptimos basados en evidencia histórica del equipo.

ILAC G-24:2022 — Guía Internacional

ILAC (International Laboratory Accreditation Cooperation) publicó la guía G-24 para orientar a los laboratorios en la determinación de intervalos de recalibración. Los principales métodos que reconoce son:

  • Método calendario: intervalo fijo en meses (ej: cada 12 meses)
  • Método por uso: basado en horas de operación o número de mediciones
  • Método estadístico: ajuste dinámico según historial de resultados de calibración
  • Método de control: revisión continua mediante comprobaciones intermedias

SAM implementa principalmente el método calendario y complementa con comprobaciones intermedias. El módulo de Intervalos analiza el historial de errores para sugerir si el intervalo debe acortarse o puede extenderse.

Módulo de Intervalos en SAM

1
Vaya a Equipos → Intervalos de Calibración
El módulo analiza el historial de las últimas confirmaciones metrológicas
2
Revise la gráfica de tendencia de errores
Si los errores están sistemáticamente lejos del EMP (zona segura), el intervalo puede extenderse. Si se acercan al límite, debe acortarse.
3
Justifique y apruebe el cambio
Todo cambio de intervalo queda registrado con su justificación técnica y la firma del responsable metrológico
Requisito de acreditación

Los organismos de acreditación (ONAC, ENAC, A2LA, etc.) verifican que los intervalos estén documentados y justificados. SAM genera automáticamente el registro de justificación que exige el evaluador.

10 Baja de Equipos

Retire equipos del inventario activo de forma trazable

¿Cuándo dar de baja un equipo?

  • El equipo está dañado y no tiene reparación viable
  • El equipo fue reemplazado por uno nuevo
  • El equipo ya no es necesario para los procesos de medición
  • El equipo fue robado o perdido
1
Abra el equipo → opción "Dar de Baja"
Solo disponible para roles Administrador y Gerencia
2
Seleccione el motivo y fecha de baja
El motivo queda registrado en el historial del equipo
3
El equipo queda inactivo
No aparece en el inventario activo, pero todo su historial se conserva para auditorías futuras. Los datos nunca se eliminan.
Trazabilidad garantizada

SAM nunca borra el historial de un equipo dado de baja. Si un organismo de acreditación solicita documentos de un equipo retirado hace 5 años, puede consultarlos en Equipos → Equipos Inactivos / Bajas.

11 Preguntas Frecuentes

Respuestas a las dudas más comunes

Acceso y Cuenta

¿Olvidé mi contraseña, qué hago?

En la pantalla de inicio de sesión haga clic en "¿Olvidaste tu contraseña?". El sistema enviará un enlace de restablecimiento al correo registrado. Si no tiene correo configurado, contacte a su administrador de sistema.

¿Por qué no veo todos los equipos?

SAM es multiempresa. Solo verá los equipos de la empresa a la que pertenece su usuario. Si necesita acceso a otra empresa, solicítelo al administrador principal.

Calibraciones y Confirmaciones

¿Puedo registrar una calibración sin adjuntar el PDF del certificado?

Sí, el PDF no es obligatorio en el formulario. Sin embargo, adjuntarlo es una buena práctica y recomendado para auditorías. Puede cargarlo después editando el registro.

¿Cuántos puntos de medición debo ingresar en la confirmación?

Depende del uso del equipo. Se recomienda al menos 3 puntos representativos del rango de trabajo: mínimo, medio y máximo. Para equipos críticos se recomiendan 5 o más puntos.

¿Qué pasa si el equipo es APTO en algunos puntos y NO APTO en otros?

El sistema marcará la confirmación como NO APTO si algún punto supera el EMP. En ese caso puede documentar que el equipo es apto para un rango de uso reducido (excluyendo los puntos no conformes), siempre que la restricción quede anotada.

Reportes y Documentos

¿Los PDFs generados tienen validez para una auditoría?

Sí. Los reportes incluyen número de registro, fecha, técnico responsable y están diseñados para cumplir los requisitos documentales de ISO 17020 e ISO 17025. Para firmas digitales, contáctenos para integrar con su sistema de firma electrónica.

El ZIP tarda mucho en descargarse, ¿es normal?

Los ZIP se generan en segundo plano porque pueden contener muchos documentos. Recibirá una notificación cuando esté listo. Si después de 15 minutos no recibe la notificación, reintente o contacte soporte.
